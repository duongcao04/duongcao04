# Project Overview

- **Name**: Yangis Shop
- **Description**: Yangis Ecommerce is a modern e-commerce platform built to facilitate the online purchase and sale of electronic devices. The platform provides a seamless user experience for both regular users and administrators.
- **Objective**: To create a user-friendly, secure, and scalable e-commerce website to manage product catalogs, orders, payments, and customer interactions.

# Features

1.  **User site**
    - Browse products by category or search.
    - View detailed product information (e.g., specifications, reviews).
    - Add products to the cart and proceed to checkout.
    - Secure payment gateway integration.
    - Order history and status tracking.
2.  **Admin Site**
    - Manage product catalogs (add, update, delete products).
    - Track and manage orders.
    - Analyze sales and user engagement through dashboards.
    - Handle promotional campaigns and discounts.

# Technologies Used

- **Frontend:** React.js, TailwindCSS, Axios, Framer Motion
- **Backend:** Nest.js, TypeORM, Jsonwebtoken, Bcrypt.
- **Database:** PostgreSQL.
- **Additional Tools:**
    - **Payment Gateway:** Momo
    - **Hosting & Deployment:** Vercel, AWS S3

# Key Contributions

- Designed and implemented a responsive UI for both user and admin interfaces.
- Developed backend APIs to manage data flow between the frontend and database.
- Integrated third-party services like payment gateways and authentication systems.
- Built a dynamic admin dashboard for tracking performance and managing operations.
- Implemented robust security features to protect user data and transactions.

# Outcomes and Results

- Successfully launched Yangis Ecommerce, catering to a wide customer base.
- Improved conversion rates by 20% due to intuitive UI and smooth UX.
- Efficient admin panel reduced order processing time by 30%.
